Instructions for Using the Enrichment Bubble Plot Shiny Application

1. Upload Data

  • Upload CSV File:
    • Click the “Choose CSV File” button to upload a CSV file containing your enrichment data. The file should include columns for Group, Description, Count_all, and Pvalue.

2. Configure Plot Settings

  • Number of Entries per Group:
    • Enter the number of top entries to display per group in the “Number of Entries per Group” field.
  • File Options:
    • Header: Check the “Header” box if your CSV file includes column headers.
  • Customize Plot Appearance:
    • Plot Title: Enter a title for the plot in the “Plot Title” field.
    • X-axis Label: Specify the label for the X-axis.
    • Y-axis Label: Specify the label for the Y-axis.
  • Color Settings:
    • Number of Colors: Specify the number of colors you want to use for the plot.
    • Colors: Use the color pickers to choose the colors for the bubbles. The number of color pickers will match the number of colors specified.
  • Plot Dimensions:
    • Plot Width (inches): Enter the desired width of the plot in inches.
    • Plot Height (inches): Enter the desired height of the plot in inches.
  • File Type for Download:
    • Select the file format for downloading the plot from the “File Type” options. Available formats include PNG, SVG, and PDF.

3. Generate and View Plot

  • Generate Plot:
    • The plot will be automatically generated and displayed in the main panel based on the settings you’ve configured.

4. Download Plot

  • Download Plot:
    • Click the “Download Plot” button to save the generated bubble plot in the selected file format. The file will be named with a timestamp to ensure uniqueness.
